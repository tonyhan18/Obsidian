# 20260406 기술 학습 요약

## 1. Radix UI & DRM 정책 대응

• 문제: Radix UI `Tabs.Content`의 `role="tabpanel"` 속성이 사내 DRM 보안 정책과 충돌함.  

• 해결: `asChild`와 `forwardRef`를 사용하여 속성을 가로채는 래퍼 컴포넌트(`RoleStripper`)를 설계, 불필요한 `role` 속성을 제거함.

• 주의: `asChild` 사용 시 단일 자식 요소만 허용하는 `React.Children.only` 에러 대응 (단일 컨테이너로 감싸기).


## 2. [Next.js](http://next.js/) + SCSS Module 최적화

• 구조 설계: `assets/css`에 모아두는 방식 대신 컴포넌트 단위로 `.module.scss`를 두는 Co-location 패턴 학습.

• 클래스 충돌 방지: 서로 다른 컴포넌트에서 동일한 이름(`.depth5`)을 사용해도 해시 기반 지역화를 통해 충돌을 원천 차단함.

• clsx 활용:
  • 객체 문법 사용 시 계산된 속성명 `[styles.active]` 적용 필수.
  • 케밥 케이스(`styles['is-active']`) 호출 및 `undefined` 무시 특성 파악.
