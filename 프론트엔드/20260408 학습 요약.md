## 1. 프론트엔드 개발 및 API 연동
• Radix UI Accordion 이슈 해결: `Accordion.Trigger` 내부의 중첩 버튼 문제를 `Accordion.Header`에서 형제 요소로 분리하고 `e.stopPropagation()`을 사용하여 해결함.
• Firebase Auth & Google 연동: NextAuth 대신 Firebase Authentication을 사용하여 구글 로그인을 구현하고, Firestore 보안 규칙과 연동하는 효율적인 아키텍처 학습.
• Google Calendar API: 개발 모드에서의 OAuth 세팅, '테스트 사용자' 등록 방법, 그리고 민감한 권한(Sensitive Scope) 요청 시 발생하는 보안 경고 해결 방안 검토.
